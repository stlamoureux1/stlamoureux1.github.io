Website Revamp

/ February 4, 2026

Welcome to the new version of my personal site! I decided to transition away from jekyll, the default build system for github pages and use hugo instead.

So far the experience has been a lot better, in my opinion. Build times are way faster, and hugo’s server utility and draft system are really user friendly for making modifications and checking them.

This site was made using the Moonwalk theme, with some slight modifications.
